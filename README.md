## What I write about

- **Arabic-first AI** — why most "Arabic LLM tools" produce stilted Egyptian/Levantine output, and what to do about it
- **Najdi dialect modeling** — making Saudi-native voice the default, not a translation target
- **Agent-native architecture** — software where the AI agent is a first-class operator, not a chatbot bolted on
- **Compound engineering** — letting AI tools build their next version, self-rewriting prompts, evolutionary content systems
- **Offshore building** — what working from an Arabian Gulf platform teaches you about software resilience
- **Saudi tech independence** — building products in the Kingdom, in Arabic, on local infrastructure where possible

Read more at [**alkinani.live**](https://alkinani.live).
